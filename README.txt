=============================================================
             NETWORK BARCODE SCANNER - PC SERVER
=============================================================

This server receives barcode/QR scans sent from your mobile app 
and types them instantly into whichever input field or text box 
is active on your computer.

-------------------------------------------------------------
HOW TO RUN THE SERVER:
-------------------------------------------------------------

1. Make sure you have Python installed on your computer.
   (Download it from https://www.python.org if you don't have it)

2. (Optional but Recommended) Open your Command Prompt / Terminal 
   and install 'pyautogui' for automatic typing:
   
   pip install pyautogui

3. Double-click "pc_server.py" or run it from your command line:
   
   python pc_server.py

4. The server will start and show your local IP Address.
   Open the mobile application, go to Settings, scan the QR code
   (or enter the IP Address manually), and connect instantly!

=============================================================
Features:
- Fast typing into any software (Excel, Word, Notepad, Web Browsers)
- Auto-Submit / Action Chain support (Tab, Enter)
- Subnet auto-discovery and zero-config QR pairing
=============================================================
